\subsubsection{End}

The \kw{end} keyword can be used in \Poetry{}, with much the same
behaviour as in \Prose{}; upon evaluation, it causes a fizzle.
In fact, the syntax and semantics are exactly the same as well.

\begin{bnf*}
    \bnfprod{End}{
        \bnfts{\kw{end}}
    }
\end{bnf*}

\begin{prooftree}
    \AxiomC{}
    \LeftLabel{End}
    \UnaryInfC{$\Gamma\vdash\kw{end}\ : \bot$}
\end{prooftree}
